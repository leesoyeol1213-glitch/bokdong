# 테스터 안내문 (카톡·메신저에 복붙해서 발송)

> 친구 테스터용. 클라우드 로그인 출시 안내 + 최신본 받기. 필요하면 아래 문구만 복사하세요.

---

🚲 **임복동 세계일주 업데이트!** 🚲

복동이가 또 커졌어요. 테스터 여러분 덕분입니다 🙏
(플레이 링크: https://bokdong.vercel.app )

🎉 **이번 최대 업데이트 — 클라우드 로그인**
이제 진행도를 계정에 안전하게 저장할 수 있어요.
👉 **폰을 바꿔도, 브라우저를 지워도, 진행도 그대로!**

▶ **쓰는 법** (비밀번호 없이 이메일만!)
① 하단 **🔑 백업** 버튼
② 맨 위 **☁️ 클라우드 세이브**에 이메일 입력 → **📧 로그인 링크 받기**
③ 메일에 온 **링크 클릭** → 끝! (📌 안 오면 **스팸함**도 확인)
※ 다른 기기에서 **같은 이메일**로 로그인하면 이어서 플레이돼요.

📱 **앱처럼 쓰기**
브라우저 메뉴 → "홈 화면에 추가" 하면 아이콘으로 바로 실행돼요.

🔄 **새 버전 나오면**
하단에 파란 **"새 버전! 탭하여 새로고침"** 배너가 떠요. 탭만 하면 최신본!
(혹시 뭔가 이상하면 새로고침 한 번 해주세요 → 항상 최신으로 고쳐집니다)

🛠️ **그동안 고친 것들**
· 대구 도착 퀴즈 먹통 → 수정
· 가챠 전설 획득이 인벤에 안 들어오던 문제 → 수정
· 처음 시작 안내(온보딩) 추가
· 전국 레이드 보상 설명 정리

버그·의견 뭐든 편하게 던져주세요. 여러분 피드백이 복동이를 키웁니다!
같이 전국 완주해요 🚴💨
