## 4. 링크 / 업로드 방식 (택1)
- **권장 — 외부 링크 임베드**: 이 게임은 Supabase(레이드·클라우드)를 쓰므로, itch에 파일을 올리기보다 **`https://bokdong.vercel.app` 를 그대로 연결**하는 게 안정적.
  - itch 프로젝트에서 "Kind of project"를 HTML로 두고 본문에 "▶ 지금 플레이" 버튼 링크를 넣거나,
  - 또는 프로젝트를 링크형으로 만들어 Vercel 주소로 보냄.
- (대안) HTML zip 업로드: 저장소 파일 전체를 zip으로 올리면 itch에서 임베드 재생되지만, 상대경로·외부 도메인 이슈 확인 필요. 처음엔 **링크 방식이 안전**.

## 5. 스크린샷 촬영 리스트 (📱 폰에서 찍으면 제일 예뻐요)
itch는 3~5장 권장. 아래 장면을 추천:
1. **라이딩 메인 화면** — 복동이가 달리는 상단 캔버스 + 하단 스탯/버튼 (게임의 얼굴)
2. **도시 도착 + OX 퀴즈** 팝업
3. **도감 탭** — 자전거/도시/맛집 컬렉션 채워진 모습
4. **🌏 주간 전국 레이드** — "🎁 이렇게 보상받아요" 안내 + 리더보드
5. **가챠/장비** 획득 연출 (전설 god-ray 뜰 때)

> 커버 이미지(썸네일 630×500)는 `assets/icon-512.png`(복동이 달리는 아이콘)를 활용하거나, 라이딩 장면을 크롭해서 사용.

## 6. 출시 후 (5단계)
- itch 페이지 공개 → 링크를 친구/커뮤니티에 공유(첫 유입).
- PostHog 대시보드에서 **유입 → 첫날 잔존(D1) → 재방문(D7)** 관찰.
- 반응 보고 리텐션 약한 구간을 개선(= 진짜 게임을 키우는 루프).
